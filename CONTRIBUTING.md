# Contributing to Ultima_RAG

First off, thank you for considering contributing to Ultima_RAG! It's people like you that make Ultima_RAG such a great tool.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and provide steps to reproduce.
- Include information about your environment (OS, Python version).

### Suggesting Enhancements
- Open an issue to discuss your idea.
- Provide a clear description of the enhancement and why it would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints.

## Style Guide
- Follow PEP 8 for Python code.
- Use meaningful variable and function names.
- Document your code with docstrings.

## Questions?
Feel free to open an issue for any questions or project-related discussions.

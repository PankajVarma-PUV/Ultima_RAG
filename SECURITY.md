# Security Policy

## Supported Versions

We currently support the latest version of Ultima_RAG. Please ensure you are running the most recent code from the `main` branch.

## Reporting a Vulnerability

We take the security of Ultima_RAG seriously. If you discover a security vulnerability, please do NOT open a public issue. Instead, please report it via one of the following methods:

- **Private Email**: [Insert Security Contact Email]
- **GitHub Private Reporting**: Use the "Report a vulnerability" button on the GitHub Security tab if enabled.

### What to include in your report:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge receipt of your report and provide a timeline for a fix. We appreciate your help in keeping Ultima_RAG secure.
